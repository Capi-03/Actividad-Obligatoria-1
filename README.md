# 🏦 Sistema Bancario - Java POO

## 📋 Descripción

sistema bancario mediante los principios de la Programación Orientada a Objetos, implementando relaciones entre clases, herencia, composición y encapsulamiento.
El sistema permite administrar clientes y cuentas bancarias, representando una estructura similar a la utilizada por una entidad financiera.

# 🛠 Tecnologías utilizadas

* Java 21
* Maven
* Lombok

# 🧱 Conceptos de POO aplicados

✔ Encapsulamiento
✔ Herencia
✔ Polimorfismo
✔ Clases abstractas
✔ Asociación entre objetos
✔ Sobrescritura de métodos
✔ Uso de constructores
